A very simple plugin to add codes PHP, CSS and JS in Wordpress.

== Usage ==

* Add any CSS snippets to `custom/style.css`.
* Add any PHP snippets to `custom/functions.php`.
* Add any JS snippets to `custom/custom.js`.
* Activate the plugin.
